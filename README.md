# Tech_maki
### 個人制作

### Tech_maki！URL: https://thawing-hamlet-64941.herokuapp.com/

#### 概要
Instagram風SNSサービスです。  

#### 主な機能
1．ユーザー登録　　  
2．ログイン・ログアウト　　  
3．投稿  
4．投稿一覧  
5．投稿編集・削除   
6．マイページ  
7．いいね   

#### 開発環境
OS: Mac OS 10.15.7  
開発言語: Ruby2.7.2  
フレームワーク: Rails 6.0.3.4  

#### 本番環境  
Heroku
